# Project Purpose

This repo is the backend foundation for YummyDoors.

Primary responsibilities:

- issue and validate auth sessions
- store customer profile and saved addresses
- power customer discovery, restaurant detail, menus, favorites, reviews
- power merchant workspaces and merchant restaurant operations
- power admin approval, category, restaurant, promo, and reservation operations
- expose stable API contracts for desktop, admin, and later mobile wiring

This repo is not the UI surface. UI work belongs in sibling repos:

- `../yummydoors_desktop`
- `../yummydoors_admin`
- `../yummydoors_mobile`
